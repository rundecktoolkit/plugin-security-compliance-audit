## Release Notes

### v1.0.1

Documentation refresh release for the Rundeck Security & Compliance Audit Plugin.

### Included

- Added README screenshots for:
  - Security & Compliance scan page
  - Scan results with live findings
  - AI report preview
  - Project configuration tabs
  - Security & Compliance settings
  - Security rule controls and severity overrides
  - AI Connection settings
- Improved public repository presentation for install and evaluation
- Added release-notes entry for v1.0.1

## Installation

1. Download `plugin-security-compliance-audit-1.0.1.jar` from this release.
2. Upload it via **System Menu -> Plugins -> Upload Plugin**.
3. Restart Rundeck if your deployment does not hot-load UI plugins.

## Notes

- This release focuses on repository and release documentation.
- The installable plugin JAR remains compatible with Rundeck 5.x.
